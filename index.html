Plottable.js (github)

Plottable.js is a charting / data visualization library built on top of D3.js. It is Palantir's most prominent open-source project. It produces charts by combining small, modular components within a layout engine. Plottable is entirely in Typescript.

Plottable is under active development by a team at Palantir, and is used in a variety of Palantir products. It is also one of the most-starred Typescript projects on Github.

I was the tech lead on Plottable from Janurary 2014, when I wrote the first line of code until November 2014, when I left Palantir. I was responsible for the architecture and many of the technical decisions. You can watch a tech talk at JSConf EU in which I describe the architecture and history of Plottable.

EvolutionSimulator, an artistic exploration of evolution and natural selection (github, full-sized)

A web-based simulation of evolving bacteria-like organisms that simulates natural selection and evolution. Built as an art project using Processing.js.

It uses web workers to run the simualtion model in a non-blocking fashion as a seperate process, and then passes model states to a renderer running on the main event loop. The front-end interpolates model updates so that even as the model slows down, the framerate remains constant. The blobs themselves have a simple genetic structure that encodes their properties and abilities, and a simple neural network that functions as an AI.

  • Worked with Professor Risi Kondor to implement the Hierarchical Bhattacharrya Kernel, a novel recursive similarity measure between trees of data.
  • First implemented the Bhattacharrya Kernel, a similarity measure between sets of data that operates by mapping datasets into an infinite-dimensional Hilbert space, fitting normal distributions to the data, and then computing the Bhattacharrya distance between the distributions.
  • Then recursively generalized the Bhattacharrya Kernel to operate on trees of sets of data, by using the Bhattacharrya Kernel as its own base kernel for comparisons.
  • Over the course of the project, I discovered and corrected several mathematical errrors in the original formulation. The code (Matlab and Python) can be found on Github.
