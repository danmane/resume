UChicago CS Dept., Research Assistant in Machine LearningJune - August 2012

  • Worked with Risi Kondor at the UChicago Machine Learning department to implement the Hierarchical Bhattacharrya Kernel, a novel recursive similarity measure between trees of data.
  • Implemented the Bhattacharrya Kernel, a similarity measure between sets of data that operates by mapping datasets into an Hilbert spaces, fitting normal distributions to the data, and then computing the Bhattacharrya distance between the distributions.
  • Recursively generalized the Bhattacharrya Kernel to operate on trees of sets of data, by using the Bhattacharrya Kernel as its own base kernel for comparisons.
  • Discovered and corrected several mathematical errrors in the original formulation.
  • The code (Matlab and Python) is on Github.
